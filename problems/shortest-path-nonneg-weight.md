# Problem
Let $G=(V,E)$ be a simple graph, and $w:E \to S$ be a weight function, where $S$ is an ordered semigroup with no negative elements. Find a shortest path between $s,t\in V$ with minimum weight.

# Algorithms

## Dijkstra's algorithm

- $O(m+n\log n)$ time.
- $O(n)$ space.
